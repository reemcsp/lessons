# Bash

## Change folder location
Move forward
```bash
cd folder_name/sub_folder_name
```
Move backwards
```bash
cd ..
```

## List all files in folder

Basic `ls`, no extra data
```bash
ls 
```
Advanced `ls`, more data
```bash
ls -la
```

## Remove Stuff
Remove file
```bash
rm file_name
```
Remove Folder
```bash
rm -rf folder_name
```

## Create File
```bash
touch file_name.file_format
```

## Clear Terminal
```bash
clear
```

## Move folder
```bash
mv source target
```

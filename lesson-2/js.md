# javascript

js files end with `.js`

## variables
const - cant be changed
```javascript
const GREETING_ENGLISH = "hello";
const GREETING_HEBREW = "shalom";
```
let - can be changed
```javascript
let count = 0;
```

## Objects
```javascript
const my_object = 
    {   
        sunday: 0,
        monday: 1,
        ...
    }
```
